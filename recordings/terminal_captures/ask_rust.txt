**What is Rust?**

Rust is a systems programming language developed by Mozilla, first released in 2010. It focuses on three core goals:

- **Safety** - Memory safety without garbage collection through its ownership and borrowing system
- **Concurrency** - Built-in thread safety guarantees to prevent data races
- **Performance** - Zero-cost abstractions and control over low-level details

Common uses include:
- Systems programming
- WebAssembly applications
- Command-line tools
- Web services and APIs
- Embedded systems development

**Cargo** is Rust's package manager and build tool, handling dependency management and compilation.
