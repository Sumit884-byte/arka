Free AI credits — quick setup guide
========================================

Your providers
  ✓ Google Gemini      configured
  ✓ Groq               configured
  ✓ OpenRouter         configured
  ✓ Ollama (local)     configured

Your settings
  Config file:     ~/dev/arka/.env
  ROUTE_MODE:      ai_only
  Preferred LLM:   openrouter → x-ai/grok-4.20-multi-agent
  Key rotation:    on
  Auto failover:   on

1. Get free API keys
   1. Google Gemini: https://aistudio.google.com/apikey
      Generous free tier; default failover chain starts here.
   2. Groq: https://console.groq.com/keys
      Fast Llama models with a free tier.
   3. OpenRouter: https://openrouter.ai/keys
      One key unlocks many models, including :free variants.
   Local option: install Ollama and pull a small model (llama3.2:1b, qwen3:8b).

2. Add keys to your .env
   Edit: ~/dev/arka/.env
   Minimum (pick one or more):
     GEMINI_API_KEY=***
     GROQ_API_KEY=***
     OPENROUTER_API_KEY=***
   Stretch free quota with backup keys:
     GEMINI_API_KEYS=key2,key3
     GROQ_API_KEY_2=backup_key

3. Save tokens on routing
   Add to .env:
     ROUTE_MODE=symbolic
     LLM_AUTO_FALLBACK=1
     API_KEY_ROTATION=1
   symbolic routes most skills locally; the LLM only picks skill names.

4. Tune models for your machine
   arka select best model for my pc
   select_model --apply
   Or set: AI_PREFERRED_PROVIDER=gemini
           AI_PREFERRED_MODEL=gemini-2.0-flash

5. Verify setup
   arka setup          # venv + dependencies
   arka doctor         # providers, fish, venv
   arka ai-models      # live model list

Tips
  • Failover order: Gemini Flash → Groq Llama → Ollama (see docs/concepts/llm).
  • OpenRouter free models include google/gemini-2.0-flash-exp:free.
  • 3D models: TRIPO_API_KEY (300 free credits) or HF_TOKEN for Shap-E Space — compose_3d check
  • Context7 (CONTEXT7_API_KEY) is optional MCP docs lookup — not your chat LLM.
  • Reset exhausted models: python3 arka_llm.py reset-exhaustion
