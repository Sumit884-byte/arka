`arka` 命令是主入口。`agent` 是同一路由器的别名。

## 主要命令

| 命令 | 说明 |
| ---- | ---- |
| `arka <request>` | 将自然语言路由到最佳技能 |
| `agent <request>` | 相同路由器（别名） |
| `arka ask <question>` | Web + AI 回答 |
| `arka route <request>` | 预览路由但不执行 |
| `arka goal <goal>` | 自主多步骤 agent 循环 |
| `arka setup` | 初始化配置目录和依赖 |
| `arka doctor` | 安装诊断 |
| `arka reload` | 重新读取 `.env` 和 `config.fish` |
| `arka refetch --install` | Git pull + sync + 重新安装 |
| `arka help` | 技能概览 |

## Goal agent

以步骤预算运行自主多步骤任务：

```bash
arka goal set up a venv, install requests, and run pytest
arka goal -y -n 30 debug why nginx fails
```

默认步骤预算：`GOAL_MAX_STEPS=25`。

## Fish shell 辅助函数

使用 fish 时，可用以下函数：

```fish
agent_route "play bohemian rhapsody"   # 预览路由
agent_trace                            # 上一次路由决策
agent_why                              # 解释技能选择
```

## 应用配置更改

```bash
arka reload              # 重新读取 .env + config.fish
arka reload --listen     # 同时重启唤醒监听器
```
