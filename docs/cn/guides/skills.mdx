## 图表

完整指南：[图表与图形](/cn/guides/charts)

生成 matplotlib PNG 图表并保存到 `~/Pictures/arka-generated/`：

```bash
chart line AAPL TSLA --range 1y
arka phone sales Apple 230 Samsung 210 Xiaomi 140
```

支持的类型：折线图、条形图、饼图、散点图、直方图、帕累托图。
