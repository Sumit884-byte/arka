## Charts

Full guide: [Charts and graphs](/guides/charts)

Generate matplotlib PNG charts to `~/Pictures/arka-generated/`:

```bash
chart line AAPL TSLA --range 1y
arka phone sales Apple 230 Samsung 210 Xiaomi 140
```

Supported types: line, bar, pie, scatter, histogram, pareto.
