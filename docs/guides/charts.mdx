Turn numbers into PNG charts. Seven chart kinds supported via offline NL parsing — no LLM required for most requests.

```bash
pip install "arka-agent[charts]"
```

## Chart types

| Kind | CLI | NL example |
| ---- | --- | ---------- |
| line | `chart line AAPL TSLA --range 1y` | `compare Apple and Tesla stock last year` |
| bar | `chart bar --data "A:1,B:2"` | `phone sales Apple 230 Samsung 210` |
| pie | `chart pie --data "A:40,B:30"` | `pie chart traffic organic 400 direct 300` |
| scatter | `chart scatter --data "100:200,120:190"` | `scatter ad spend vs revenue 100 200 120 190` |
| histogram | `chart histogram --data "12,15,18,22"` | `histogram response times 12 15 18 22` |
| pareto | `chart pareto --data "A:45,B:28"` | `pareto defects Scratches 45 Dents 28` |

## Direct CLI

```bash
chart line AAPL TSLA --range 1y
chart bar --data "Apple:230,Samsung:210,Xiaomi:140" --title "Phone sales"
chart pie --data "Organic:400,Direct:300,Referral:300" --title "Traffic"
```

## Chart defaults

Store reusable data in `~/.config/arka/charts.yaml`:

```yaml
defaults:
  scatter:
    data: "100:200,120:190,170:280"
    xlabel: "Ad Spend"
    ylabel: "Revenue"
  line:
    tickers: [AAPL, TSLA]
    range: 1y
```

Charts save to `~/Pictures/arka-generated/` and open automatically on macOS/Linux.

```env
# CHART_OUTPUT_DIR=~/Pictures/arka-generated
# CHARTS_CONFIG=              # override defaults file path
```
