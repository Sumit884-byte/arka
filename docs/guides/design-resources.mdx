---
title: "Default design resources"
description: "Use a curated developer design-resource catalog safely."
---

Arka uses [Design Resources for Developers](https://github.com/bradtraversy/design-resources-for-developers)
as its default discovery catalog for design workflows.

```bash
arka design-resources
arka design-resources prompt
```

The catalog is reported as MIT-licensed, but every linked font, icon, template,
or asset must be checked independently before production use.
