# Arka Ideate

Use Arka Ideate to explore open-source momentum before building:

```bash
arka ideate "open-source AI developer tools"
arka ideate "trending frontend infrastructure" --deep
```

The workflow asks Arka to check current GitHub/package/discussion signals,
licenses, maintenance, open issues, and integration friction, then identify an
underserved problem and suggest a concrete improvement. Use live web or MCP
research for evidence; popularity alone is not product validation.
