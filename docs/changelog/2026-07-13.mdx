## New features

- **CSV/TSV table previews over MCP.** Connected agents can now request clean, structured previews of CSV and TSV files through Arka's MCP server — no ANSI escape codes or terminal formatting in the output. This makes it easier for external AI clients (Cursor, Claude Desktop, and other [MCP-compatible tools](/guides/mcp)) to inspect tabular data alongside Arka's [data Q&A skills](/guides/data-ask).
